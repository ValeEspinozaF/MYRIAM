      Remarks

      If nBins nor binEdges paramaters are supplied, the grid will be divided in 50 bins on each dimension. Also, if both binEdges and range parameters are given, range is ignored.

      Exceptions

      ArgumentExceptionThrown when:

      Arrays arrayX and arrayY do not have the same length.

      Inputs nBins and binEdges are both supplied.

      Input nBins is empty or has more values than 2 values (one for each dimension).

      Input nBins must be a positive integer bigger than zero.

      Input binEdges values must be in increasing order.

      Input range values must be increasing in each dimension.
