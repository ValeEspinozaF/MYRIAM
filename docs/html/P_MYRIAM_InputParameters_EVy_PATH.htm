      InputParametersEVy_PATH Property

      String path to a 3-column plain-text file containing an ensemble of sampled Cartesian values of the younger Euler vector, expressed in deg/Myr (degrees per million year). Alternatively, MYRIAM can sample its own ensemble from a singular Euler vector stage, by supplying a 10-column single line containing: [1--2] Euler pole longitude and latitude in degrees, [3] angular velocity magnitude in deg/Myr, [4--9] elements of the covariance matrix associated with the Euler vector ensemble, in rad^2/Myr^2, and [10] size of the ensemble.
