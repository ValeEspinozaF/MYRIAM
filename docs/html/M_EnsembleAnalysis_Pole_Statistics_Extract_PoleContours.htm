      Remarks

      This function takes 2 arrays of xy coordinates and turns them into a 2D histogram (MakeHistogram) with bins that record the spatial recurrence of coordinates. This recurrence is now the Val value, whereas the 2D-bin center is the Lon an Lat coordinates. The CreateContour method then finds the level for each given confidence interval percentage, i.e. the Val-value that encloses where a given percentage of most-recurrent samples fall and inner function CreateIsobar returns the coordinates for that z-value isobar.
